# Art Gallery Event Webpage

This is a simple HTML-based webpage created to showcase an art gallery event. The project is designed for learning and practice purposes in basic web development.

## Features
- Displays upcoming event details
- Includes gallery images
- Provides event description
- External link to explore more art
- Contact information section

## Technologies Used
- HTML

## Project Purpose
This project was created to understand basic HTML structure, including headings, images, links, and layout design.
